# OKExPyWebsocket
Python Websocket连接OKEx做量化交易框架， 注REST API和Websocket都需要Jump over Wall.



**依赖相关**

Python >= 3.6,  `pip install -r requirements.txt`



**填写个人信息配置**

`config.py`

```
API_KEY = "*"
SECRET_KEY = "*"
PASSPHRASE = "*"
```
